Welcome to my painting program!

First what's awsome?

  • Can set the color for the canvas.
    • Clicking the 'Change Canvas Color' does not delete shapes that were palced on screen. Need to click 'Clear canvas' to actually clear everything.
    • Clearing canvas doesn't reset the canvas color to the default black color, it stays on the same color user picked.
  • Can see the rgba values when changing the color sliders.
  • Two new modes:
    • Drag mode where you can drag individual shapes across the canvas with the mouse.
      • Know that you can not place shapes if this mode is checked.
      • Note: One issue occurs when moving the mouse too fast. Can lose hold of the shape or grab other shapes.
      • See README.md for more information.
    • Delete mode where you can delete individual shapes on the canas.
  • For shapes and the canvas you can change the alpha value and thus the transparency.
  • You can scale all shapes (Squares, Triangles, and Circles) by adjusting size slider but also by inputing a scaler value for the x or y axis individually.
    • Range is arbitrarily set between 0.5 and 5
    • Going below 1 shrinks the shape, e.g. 0.5 creates a shape half as big, relative to size. If size = 10, scale x = 0.5, final shape will have a dimension where x = 5 and y = 10.
    • Default scale at 1 for both x and y.
  • When painting (dragging mouse) there is a minimum distance before a new shape is created.
